# React19 Demo

安装 node 和 npm

- `npm install`：安装依赖（你会经常看到项目根目录有个`package.json` ， `npm install` 就是根据这个文件把项目需要的东西都装好)
- `npm install xxx` ：安装某个库，比如： `npm install axios`
- `npm run dev` ：启动项目，开启本地开发服务器
- `npm run build：`打包项目，准备上线
